### Prerequisites

- pre-commit >= 3
- Earthly >= 0.7
- Node.js >= 20
- pnpm >= 8

### Installing

- Install pre-commit hooks

```

pre-commit install

```

- Install Node.js dependencies

```

pnpm install

```
